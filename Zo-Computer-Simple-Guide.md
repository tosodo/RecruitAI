### 4. Skills (Superpowers) вҡЎ
Skills are **extra abilities** Zo can learn. Here's what you have:

| Skill | What it does | Use when... |
|-------|-------------|-------------|
| `demo_generation` | Creates example websites to show value | Need to show someone a sample website |

| `website_audit` | Checks websites for problems | Want to improve a website |
| `skill_optimizer` | Makes other skills work better | A skill isn't working well |
| `zopenclaw` | Runs OpenClaw (AI agent framework) | Want to run a persistent AI agent |
| `supermemory` | Helps Zo remember things better | Need long-term memory |

**When to use each skill (Simple version):**

| Check if a website is good | `website_audit` |
| Make a sample website fast | `demo_generation` |
| Have Zo remember things forever | `supermemory` |
