# CLIENT ONBOARDING CHECKLIST
> Follow this checklist for every new AIGENTFORCE client

---

## PHASE 1: WELCOME (Day 0)

- [ ] Send welcome email FROM sasha@aigentforce.io
- [ ] Schedule discovery call via Calendly
- [ ] Create client brief in /home/workspace/data/client-briefs/[slug].md
- [ ] Add client to leads.json with tier, source, contact info

---

## PHASE 2: DISCOVERY CALL (Day 1-3)

- [ ] Conduct 30-minute discovery call
- [ ] Document: biggest challenge, current tools, team size, budget, timeline
- [ ] Identify: 3 quick wins (things that take <1 hour to fix)
- [ ] Identify: 1 big opportunity (things that generate £1000+/month)

---

## PHASE 3: AUDIT & PROPOSAL (Day 4-7)

- [ ] Run website audit (Skills/website_audit/)
- [ ] Generate audit report PDF
- [ ] Create proposal with: quick wins + big opportunity + pricing
- [ ] Send proposal FROM larry@aigentforce.io
- [ ] Follow up in 48 hours

---

## PHASE 4: BUILD (Day 8-45)

- [ ] Client signs up
- [ ] Set up n8n workflows for client
- [ ] Configure email integration (sales@[clientdomain].io)
- [ ] Build chatbot if applicable
- [ ] Set up Buffer/LinkedIn if applicable
- [ ] Test all automations
- [ ] Document setup in client brief

---

## PHASE 5: DELIVER & GROW (Ongoing)

- [ ] Send weekly status report (Alex format)
- [ ] Review monthly: whats working, whats not
- [ ] Identify upsell opportunities
- [ ] Request testimonial/referral
